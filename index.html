About This Platform

Purpose & Scope

This is an educational molecular docking platform designed to demonstrate computational biology concepts in an accessible, browser-based environment. All computations run locally using JavaScript and Web Workers‚Äîno data is sent to external servers. The platform combines protein-protein docking (antibody-antigen), small molecule docking, and batch virtual screening capabilities.

Key Features

‚Ä¢ Antibody-Antigen Docking: Rigid-body docking with parallel sampling across multiple CPU cores

‚Ä¢ Batch Screening: Screen up to 50 SMILES against 2-3 receptor structures with enthalpy calculations

‚Ä¢ Scoring Function: Van der Waals-aware contact scoring with clash detection, H-bond detection, salt bridge identification, and electrostatic contributions

‚Ä¢ Enthalpy Estimation: Approximate binding enthalpies (ŒîH) using weighted energy terms

‚Ä¢ Reproducibility: All results include parameter snapshots and random seeds for exact replication

‚Ä¢ Quality Assurance: Built-in QA mode tests reproducibility across multiple runs

‚Ä¢ Session Management: Save and restore complete docking sessions with all data

Technical Details

Sampling: Uniform random sampling of SO(3) rotations using quaternions, with random translations within specified radius

Scoring: Grid-accelerated contact detection, soft-core van der Waals potential for clashes, geometric H-bond and salt bridge detection

Parallelization: Web Workers distribute sampling across available CPU cores (typically 4-8 workers)

Visualization: 3Dmol.js library for interactive molecular graphics with cartoon, stick, and surface representations

File Formats: Standard PDB for proteins, SMILES for ligands (with simplified 3D generation)

Enthalpy Calculation Methodology

The binding enthalpy (ŒîH) is estimated using a weighted sum of interaction energy terms:

ŒîH ‚âà ‚àí0.5 √ó (favorable contacts) ‚àí2.5 √ó (hydrogen bonds) ‚àí4.0 √ó (salt bridges) ‚àí1.0 √ó (hydrophobic contacts) +0.3 √ó (clash penalty) +1.0 √ó (electrostatic repulsion) +0.2 √ó (desolvation penalty) Units: kcal/mol These weights are derived from statistical analysis of experimental binding data but represent approximations for educational use.
