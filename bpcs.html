Backbone Restraints Script

Sets B-factor to 0 for all atoms, then 2.0 for protein backbone. Save as a script and run in your preferred visualization tool.

# Select all atoms and zero B-factor
set sel [atomselect top "all"]
$sel set beta 0.0

# Protein backbone
set bb [atomselect top "protein and name N CA C O"]
$bb set beta 2.0

# Write out modified PDB
$sel writepdb restraints.pdb

Quick Cartoon + Ligand View

Creates a publication-style cartoon for protein and licorice for ligand.

# Load topology/coords
mol new system.psf
mol addfile system.pdb

# Clean default rep
mol delrep 0 top

# Protein cartoon
mol representation NewCartoon 0.3 10.0 4.0 0
mol color Structure
mol selection {protein}
mol addrep top

# mol selection {resname LIG}
# mol representation Licorice 0.3 12.0 12.0
